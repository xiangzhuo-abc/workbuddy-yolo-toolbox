# 更新日志

本项目遵循语义化版本。公开版本同时提供对应源码、Windows 二进制文件和 SHA256。

## 未发布

### 新增

- 增加按需机器学习运行时管理，统一承载训练、测试、评估、自动预标注和 TensorBoard。
- 增加 CPU 与 CUDA 11.8 两种独立 Worker 运行时、可信下载清单、断点续传、离线导入、安全解压和原子安装。
- 增加运行环境管理界面，以及只查询 `py -0p` 和 `PATH` 的外部 Python 高级检测。
- 增加主界面“停止训练”控制，确认后终止训练并保留最近完成轮次的权重。

### 调整

- Windows 基础安装包改为单 GUI EXE，明确排除 Torch、Ultralytics、TensorBoard、Polars 和 CUDA DLL，目标体积不超过 300 MiB。
- 模型功能启动前统一解析并复验外置 Worker；基础标注和数据集工具不再依赖机器学习运行时。
- 发布构建拆分为基础程序、CPU 运行时 ZIP、CUDA 11.8 运行时 ZIP 和基础安装器，升级时复用兼容运行时。
- 训练窗口优先选择轻量 `yolov8n.pt`，并对中型及大型模型显示耗时和显存提示。

## 0.9.0-beta.1 - 2026-07-20

### 新增

- 确立 AGPL-3.0-only 开源发布策略。
- 集中管理产品名、显示版本和 Windows 文件版本。
- 增加第三方软件声明和 Windows EXE 产品化设计。
- 增加 PyInstaller 便携构建、Inno Setup 安装器和 Windows 版本资源。
- 增加不包含用户数据的诊断包、全局崩溃报告和关于入口。
- 将 CUDA 11.8 便携目录中的重复 Torch DLL 自动去重，降低发布体积。

### 说明

- 此版本处于 Windows EXE 产品化阶段，不作为 1.0 稳定版承诺。
- 默认发布物不包含用户数据集、模型、训练结果、日志或本机配置。
